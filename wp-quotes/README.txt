WordPress plugin that displays a random quote from a mySQL server.

== Description ==

This is an effort to create a WordPress plugin from scratch. The purpose of the plugin is to grab a random quote from a mySQL server and display it on a page as a widget or from a shortcode.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-quotes.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
